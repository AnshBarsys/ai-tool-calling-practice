# Simple AI Agent with Tool Calling

This project demonstrates a minimal AI agent that performs **tool calling**.

## How it works
- The agent reads user input
- Decides whether a tool is needed
- Calls the tool
- Returns the result

## Tool example
- `get_random_cocktail()` returns a random cocktail

## Why this exists
Tool calling is a core concept in modern LLM-based AI agents.
This project demonstrates the logic without external frameworks.
